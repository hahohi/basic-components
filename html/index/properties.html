사용 예시
Subject 심볼 아래에 CSS Code를 나열한 기본적인 형태입니다.
컨디션별 혹은 테마 구분 없이 전체적으로 적용되는 코드만 나열되어 있는 경우에 사용합니다.
기본형에서는 General 소주제를 사용하지 않습니다.
Theme 구분 없이 Condition 별로 정의가 변경되는 경우 공통 정의는 General 이라는 소주제 아래에 입력하고, 변경 부분은 Normal(혹은 Default)과 Disabled(혹은 Dim)로 구분하여 정의합니다.
Theme 및 Condition에 대한 정의가 모두 필요한 경우, General, Light theme, Dark theme
세 소주제로 나누어 정의하고 Condition은 주석으로 간략하게 정의합니다.
Image에 대한 정의가 필요한 경우, 우측 참고용 박스와 같이 제목 아래 이미지를 위치시키고
동일하게 정의를 이어나갑니다.
Refer to each app color와 같은 코멘트가 필요한 경우, 소주제 Note 를 하단에 배치함으로써
정리합니다.
