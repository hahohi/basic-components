스크린 간 링크는 Sketch 에서 제공하는 Prototyping 기능을 사용하면 됩니다.
그러나 Responsive layout 에서 버튼을 눌러 새 창으로 나타나는 Sketch measure plugin 가이드의 연결은
외부 폴더에 있는 문서를 링크해야 하므로 Anima Plugin의 기능을 사용해야 합니다.
링크를 연결할 레이어를 선택하고 Prototype 탭을 클릭하면 나타나는 Flow 기능 중 Links를 클릭해 주세요.
외부 폴더 링크
Prototype > Flow > Links 를 선택하면 좌측 상단에 'EXTERNAL LINK' 라는 버튼이 나타납니다.
위 버튼을 클릭해 주세요.
